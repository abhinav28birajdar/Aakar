A modern mobile platform for designers to discover, share, and connect.

## ✨ Features

- **Discover** - Browse stunning designs from talented creators worldwide
- **Explore** - Search and filter by categories (Logo, UI/UX, Branding, Illustration, Web, Mobile)
- **Create** - Upload and publish your creative projects with built-in editing tools
- **Connect** - Follow designers, like projects, save favorites, and engage with the community
- **Real-time** - Instant notifications for likes, comments, and new followers
- **Themes** - Beautiful light and dark modes with automatic system detection
- **Secure** - End-to-end encrypted API key management with row-level security
- **Offline** - Continue browsing with smart caching when offline

## About

**Aakar** brings together designers from around the world to showcase their creative work and build meaningful professional connections. Built with React Native, Expo, and Supabase, it provides a beautiful, fast, and secure platform for the design community to thrive.
