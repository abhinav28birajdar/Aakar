# Aakar - Design Community Platform

  <img src="./assets/images/icon.png" alt="Aakar" width="120" />

A modern mobile platform for designers to discover, share, and connect.

## ✨ Features

- **Discover** - Browse stunning designs from talented creators worldwide
- **Explore** - Search and filter by categories (Logo, UI/UX, Branding, Illustration, Web, Mobile)
- **Create** - Upload and publish your creative projects with built-in editing tools
- **Connect** - Follow designers, like projects, save favorites, and engage with the community
- **Real-time** - Instant notifications for likes, comments, and new followers
- **Themes** - Beautiful light and dark modes with automatic system detection
- **Secure** - End-to-end encrypted API key management with row-level security
- **Offline** - Continue browsing with smart caching when offline

## About

**Aakar** brings together designers from around the world to showcase their creative work and build meaningful professional connections. Built with React Native, Expo, and Supabase, it provides a beautiful, fast, and secure platform for the design community to thrive.


Aakar aims to:
- **Empower Designers** - Provide a beautiful, intuitive platform for showcasing creative work
- **Foster Community** - Connect designers across different disciplines and skill levels
- **Inspire Creativity** - Curate the best designs and make them easily discoverable
- **Support Growth** - Help designers build their professional presence and network

## 🛠️ Built With

**Frontend**
- React Native with Expo for cross-platform development
- TypeScript for type-safe code
- Zustand for state management
- Moti & Reanimated for smooth animations
- NativeWind for utility-first styling

**Backend & Database**
- Supabase (PostgreSQL) for real-time database
- Row Level Security for data protection
- Supabase Storage for scalable image hosting
- Edge Functions for serverless logic

**Design System**
- Custom design tokens for consistency
- Reusable component library
- Accessibility-first approach
